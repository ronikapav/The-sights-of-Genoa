Данное веб-приложение представляет из себя интерактивный гид по пяти известным достопримечательностям, которые можно посетить в Генуе, Италия: Палаццо Дукале, маяк Лантерна, Аквариум Генуи, Базилика Сантиссима Аннунциата дель Вастато, Монументальное кладбище Стальено.

## Особенности приложения
- **Метод filter**: Используется метод `filter`, чтобы обеспечить интерактивную фильтрацию достопримечательностей для более удобного поиска.
- **Слайдшоу**: Предусмотрены картинки достопримечательностей, которые автоматически меняются каждые 3 секунды, и кнопки для просмотра слайдов вперед и назад.
- **Удаление посещаемых достопримечательностей**: Пользователь может удалить достопримечательность из списка, если она уже была посещена.

## Установка и запуск
1. Клонируйте данный репозиторий на ваше устройство.
2. Откройте файл `index.html` в вашем веб-браузере.
